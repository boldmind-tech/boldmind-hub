# Boldmind Hub — Codespaces Quickstart

This repo is a Next.js 15 + Tailwind CSS app.

## Quickstart (GitHub Codespaces / mobile)

- Press Run Task and choose "Dev: Next.js" (or run npm run dev).
- The dev server will auto-forward port 3000 and open a browser link.
- Edit files under `app/` and see changes live.

## Scripts

- `npm run dev` — Start Next.js dev server
- `npm run build` — Build for production
- `npm run start` — Start production server (after build)
- `npm run lint` — Lint the project

## Notes

- If the port link doesn’t open, use the Ports panel and open 3000.
- Environment variables: add a `.env.local` if needed (not committed).

## Troubleshooting

- If you see missing packages, run `npm install`.
- If the server won’t start, try killing any process on 3000 and re-run.
